有时候我们添加的一些资源，如图片和一些没用的代码，以及在添加第三方库的时候我们只需要使用其中的一部分功能和一部分资源，那么这个时候如果靠我们手工去怕是非常难做的，尤其是项目大的时候，Android 团队工程师Tor Norbye ，在2014年11月1日Google ＋上发布，说Android Gradle插件支持删除一些没有用的资源，配置方法如下：

android {
   buildTypes {
        release {
            minifyEnabled true
            shrinkResources true
        }
    }
}﻿
